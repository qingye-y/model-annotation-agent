## 一、打包前清理命令（执行顺序）

在项目目录下执行以下命令，清理临时文件：

```bash
# 1. 清理 Python 缓存
find . -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null || true
find . -type f -name "*.pyc" -delete
find blueprints/ -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null || true
find services/ -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null || true

# 2. 清理日志文件
rm -f *.log

# 3. 清理截图（可选，节省空间）
find . -maxdepth 1 -type f -name "screenshot_*.png" -delete

# 4. 清理备份文件
find . -type f -name "*.bak" -delete

# 5. 清理 Playwright 测试记录
rm -rf .playwright-cli/

# 6. 清理数据库文件（重要：不要发生产数据！）
rm -f *.db instance/*.db-journal
```

## 四、必须修改的配置（发送给接收者）

在 `config.py` 中，告知对方修改以下内容：

```python
# 1. SECRET_KEY（第 4 行）- 改为随机字符串
SECRET_KEY = 'dev-secret-key-change-in-production'
# ↓ 改为
SECRET_KEY = '任意随机字符串，例如：abc123xyz...'

# 2. IDATA_COOKIE（第 16 行）- 替换为对方自己的 Cookie
IDATA_COOKIE = "..."  # 当前 Cookie 已包含在代码中
# ↓ 替换为对方从 iData 平台获取的新 Cookie
```

**Cookie 获取方法**：
1. 登录 https://idata.cai-inc.com
2. F12 → Network → 任意请求 → Headers → 复制完整 Cookie
3. 粘贴到 config.py 第 16 行
