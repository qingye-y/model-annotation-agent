# 标注详情弹窗体验修复 — 技术方案

版本：v1.0 | 日期：2026-05-26 | 状态：已完成

## 1. 需求背景

标注员在标注任务详情弹窗中反馈两个核心体验问题：

1. **图片全部无法显示**：弹窗左侧图片区域为空白，标注员无法查看商品图片来判断模型审核结果是否正确。
2. **没有商品链接跳转入口**：标注员需要查看商品链接时无处可点，只能手动拼接或去其他系统查询，效率极低。

## 2. 问题根因

| 问题 | 根因 |
|------|------|
| 图片不显示 | ① 图片 CDN（itemcdn.zcycdn.com / sitecdn.zcycdn.com）有 Referer 防盗链；② 图片字段是逗号分隔的多 URL 字符串，前端未拆分，把整串当单个 src 塞给 `<img>`；③ 页面存在 JS 错误（toggleMoreFilters 函数报错），阻塞弹窗图片渲染逻辑 |
| 没有商品链接 | 后端 `api_my_tasks` 未返回 `product_link` 字段；前端弹窗模板未渲染该字段 |

#### 3.2.3 loadAnnotImages 函数（已有，无需修改）

合并主图 / SKU图 / 详情图，拆分为独立 URL 列表，调用 `renderAnnotImageGrid()`。

```javascript
function loadAnnotImages(item) {
  annotImageList = [];
  var sources = [
    { key: 'main_image',   label: '主图' },
    { key: 'sku_image',    label: 'SKU图' },
    { key: 'detail_image', label: '详情图' }
  ];
  sources.forEach(function(src) {
    if (item[src.key]) {
      var urls = item[src.key].split(',').map(function(u) { return u.trim(); }).filter(Boolean);
      urls.forEach(function(url) {
        annotImageList.push({ src: url, label: src.label });
      });
    }
  });
  renderAnnotImageGrid();
}
```

#### 3.2.4 renderAnnotImageGrid 函数（本次调整 v1.1）

**按来源分组展示**，每组前面插入分组标题（行 2521）：

```javascript
function renderAnnotImageGrid() {
  var grid = document.getElementById('annotImageGrid');
  grid.innerHTML = '';

  var currentLabel = null;

  annotImageList.forEach(function(img, index) {
    // 如果当前图片的标签与上一张不同，插入分组标题
    if (img.label !== currentLabel) {
      currentLabel = img.label;
      var title = document.createElement('div');
      title.className = 'annot-img-group-title';
      title.textContent = currentLabel + '：';
      grid.appendChild(title);
    }

    // 创建图片容器
    var div = document.createElement('div');
    div.className = 'annot-prod-img-item';
    div.innerHTML = '<img src="' + img.src + '" alt="' + img.label + '" ' +
                    'referrerpolicy="no-referrer" ' +
                    'onerror="this.src=\'data:image/svg+xml,...\'; this.onerror=null;" ' +
                    'onclick="openAnnotImagePreview(' + index + ')">';
    grid.appendChild(div);
  });
}
```

**分组效果**：`主图：xxx...` → 若干张主图 → `SKU图：xxx...` → 若干张 SKU 图 → `详情图：xxx...` → 若干张详情图。
