## 五、核心数据流

```
iData平台（5个实例）
    ↓ Cookie认证
Flask后端 /api/data-fetch
    ↓ SQL构建+分页+随机抽样
SQLite (raw_data)
    ↓ 关键词提取
DailyStats 统计快照
    ↓
前端页面（看板/标注/质检/Badcase）
```

**支持的5个实例**：
- ZJWC（浙江网超）- 云环境
- HWCS（浙江乐采网超）- 云环境
- HNLCWC（湖南乐采网超）- 云环境
- YNLCY（云南乐采云）- 乐采云环境
- GXLCY（广西乐采云）- 乐采云环境

## 十、交接建议

如果要他人完整复制，建议：

1. **复制整个项目目录**（包含 templates/, blueprints/, static/, instance/）
2. **确保 Python 环境**：Flask 2.3.3 + 依赖
3. **填入 iData Cookie**：在 config.py 中替换 IDATA_COOKIE
4. **启动并验证**：访问 localhost:5000 测试
