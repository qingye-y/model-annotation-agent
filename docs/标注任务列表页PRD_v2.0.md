## 1. 需求背景

标注员被分配任务后，需要在一个表格中看到所有待处理和已完成的任务。每个任务应该按「规则 + 日期 + 实例」聚合展示，包含进度、编码，方便追溯。

---

## 2. 页面布局

```
标注员工作台 → 标注任务（annotation_list.html）
    ├── 任务列表表格（默认展示）  ← 本次PRD核心
    └── 商品明细分页 → 弹窗标注
```

---

## 3. 任务列表表格

### 3.1 表格列

| # | 列 | 说明 | 来源 | 
|---|-----|------|------|
| 1 | **任务编码** | 唯一标识，格式 `ANN-YYYYMMDD-NNN` | 系统生成 |
| 2 | **审核规则** | 提示词规则名称 | `GET /api/prompt-rules` + 实例映射 |
| 3 | **数据日期** | 数据的业务日期 | RawData.created_date[:10] |
| 4 | **实例** | 实例编码 | RawData.instance_code |
| 5 | **总条数** | 该任务下商品总数 | 统计 |
| 6 | **待标注** | 尚未标注的数量 | 统计（红色高亮） |
| 7 | **已标注** | 已经标注完的数量 | 统计（绿色） |
| 8 | ✅ 正确 | 标注为"正确"的条数 | 统计 |
| 9 | ❌ 错误 | 标注为"错误"的条数 | 统计 |
| 10 | ⬜ 忽略 | 标注为"忽略"的条数 | 统计 |
| 11 | **进度** | 进度条 + 百分比 | 已标注/总条数 × 100% |
| 12 | **分配人** | 谁分配的 | dispatch_log.admin_name |
| 13 | **分配时间** | 何时分配的 | dispatch_log.created_at |
| 14 | **操作** | "进入标注"按钮 | — |

### 3.2 任务编码方案

```
格式：ANN-YYYYMMDD-NNN

组成部分：
  ANN  — 固定前缀，代表"标注任务"
  YYYYMMDD — 任务生成日期
  NNN — 三位自增序号（同一天内从001开始）

示例：ANN-20260524-001

存储位置：生成标注任务时写入 RawData 的 task_code 字段（新增）
展示位置：表格第一列，支持点击复制
```

### 3.3 规则来源修正

| 维度 | 当前（bug） | 改为 |
|------|-----------|------|
| 规则名 | `computed_error_reason`（空→回退"其他违规"） | 通过 `instance_code → instance_rule_map` 查提示词规则名 |
| 底层逻辑 | dispatch.py:api_my_tasks 行436 | 改为调用 `get_instance_rule_mapping()` 查找规则名 |

## 4. 筛选栏

表格上方筛选栏，默认显示在分组视图时隐藏；在明细视图时沿用已有的筛选栏。

| 筛选项 | 说明 |
|--------|------|
| 搜索框 | 按商品名称/ID搜索 |
| 审核规则 | 下拉，来源 prompt_rules |
| 标注状态 | 全部/待标注/已标注 |
| 实例 | 实例编码下拉 |

---

## 5. 点击进入明细

点击某行或"进入标注"按钮 → 进入该任务下的商品明细列表（复用已有表格 + 分页），面包屑显示「← 返回任务列表 | 规则名 · 日期」。

明细表格列：商品ID / 商品名称 / 类目 / 实例 / 模型A结果 / 模型B结果 / 标注状态 / 操作（标注/查看）。

---

## 6. 数据存储变更

| 字段 | 表 | 说明 |
|------|-----|------|
| `task_code` | RawData | 新增 VARCHAR(20)，生成任务时写入，格式 ANN-YYYYMMDD-NNN |

> 生成任务时（POST /api/dispatch/generate-tasks），对同一批次+规则+实例+日期的记录分配同一个 task_code。

## 8. 验收标准

| # | 验证项 | 预期 |
|---|--------|------|
| AC-1 | 表格列完整 | 14 列全部显示，无缺列 |
| AC-2 | 任务编码格式正确 | `ANN-YYYYMMDD-NNN`，同一天内自增 |
| AC-3 | 规则名正确 | 显示提示词规则名（如"浙江网超审核规则"），非"其他违规" |
| AC-4 | 进度条正确 | 百分比 = 已标注/总条数 × 100% |
| AC-5 | 操作按钮"进入标注" | 点击 → 面包屑 + 商品明细表格 |
| AC-6 | 明细表格筛选可用 | 搜索/规则/状态/实例筛选生效 |
| AC-7 | 返回任务列表 | 面包屑点"← 返回" → 回到任务表格 |
| AC-8 | 空数据 | 无任务时表格显示"暂无标注任务" |
| AC-9 | 任务编码可复制 | 点击任务编码 → 复制到剪贴板 |
