# 从零开始安装指南

> 适用对象：电脑上什么都没装、从没写过代码的人
> 版本：v1.6.0 | 日期：2026-05-27

---

## 一、首先确认你电脑上有 Python

Python 是运行这个软件的语言，先检查一下有没有。

**Mac 电脑：**

1. 打开「终端」（Terminal）：按键盘上的 `command + 空格`，输入「终端」回车

2. 在终端里输入这一行，按回车：
```
python3 --version
```

3. 看结果：
   - 如果显示 `Python 3.8.x` 或 `Python 3.9.x` 或 `Python 3.10.x` 或更高 → **有 Python，跳到第二步**
   - 如果显示 `command not found` 或报错 → **没有 Python，看下面安装**

**Windows 电脑：**

1. 按 `Win + R`，输入 `cmd`，回车，打开「命令提示符」

2. 输入这一行，回车：
```
python --version
```

3. 看结果：
   - 如果显示 `Python 3.8.x` 或更高 → **有 Python，跳到第二步**
   - 如果显示 `'python' 不是内部或外部命令` → **没有 Python，看下面安装**

---

## 二、安装 Python（如果上一步没有的话）

### Mac 安装 Python：

1. 打开浏览器，访问：https://www.python.org/downloads/mac-osx/

2. 点击黄色按钮「Download Python 3.13.x」（或最新版本）

3. 下载完后，双击打开安装包

4. **重要**：安装过程中，**一定要勾选**「Add Python to PATH」（把 Python 加入环境变量）

5. 一路点「Continue」→「Install」→ 输入电脑密码 → 等待安装完成

6. 安装完后，重新打开终端，输入：
```
python3 --version
```
看到 `Python 3.13.x` 就说明安装成功了。

### Windows 安装 Python：

1. 打开浏览器，访问：https://www.python.org/downloads/windows/

2. 点击「Download Python 3.13.x」（或最新版本）

3. 下载完 `python-3.13.x-amd64.exe` 后，双击打开

4. **重要**：安装界面第一步，**一定要勾选**「Add Python to PATH」（把 Python 加入环境变量），不勾这个后面会出错！

5. 点「Install Now」，等待安装完成

6. 安装完后，重新打开命令提示符（cmd），输入：
```
python --version
```
看到 `Python 3.13.x` 就说明安装成功了。

---

## 三、解压项目文件

1. 把收到的压缩包（`.zip` 文件）解压到你想放的地方
   - Mac：双击压缩包自动解压
   - Windows：右键压缩包 →「全部解压缩」→ 选择一个文件夹位置

2. 解压后，你会看到一个叫「模型标注agent」的文件夹

3. **记住这个文件夹在哪里**，后面要用到它的路径

---

## 四、打开终端，进入项目目录

**Mac：**

1. 打开终端（`command + 空格` → 输入「终端」）

2. 输入 `cd `（注意 cd 后面有个空格），**不要按回车**

3. 把「模型标注agent」文件夹**拖到终端窗口里**，终端会自动填入路径

4. 看起来大概是这样（路径可能不一样）：
```
cd /Users/你的用户名/Desktop/模型标注agent
```

5. 按回车

**Windows：**

1. 打开命令提示符（`Win + R` → 输入 `cmd` → 回车）

2. 输入 `cd `（注意 cd 后面有个空格），**不要按回车**

3. 在文件管理器里找到「模型标注agent」文件夹
4. 在地址栏右键 →「复制地址」

5. 粘贴到命令提示符里，看起来大概是这样：
```
cd C:\Users\你的用户名\Desktop\模型标注agent
```

6. 按回车

---

## 五、安装依赖（这一步会下载一些东西）

在终端里输入下面这行，按回车：

**Mac：**
```
pip3 install -r requirements.txt
```

**Windows：**
```
pip install -r requirements.txt
```

你会看到终端开始显示一些下载进度，这是正常的。等它跑完（通常1-3分钟）。

如果最后显示 `Successfully installed ...` 或者回到了可以输入新命令的状态，说明安装成功了。

---

## 六、修改配置文件（第1次运行必须做）

这一步很重要！不修改的话软件用不了。

1. 用记事本（Mac 用 TextEdit）打开「模型标注agent」文件夹里的 `config.py` 文件

2. 找到这两行：

```python
SECRET_KEY = 'dev-secret-key-change-in-production'
IDATA_COOKIE = "_zcy_log_client_uuid=..."
```

3. 把第一行改成：
```python
SECRET_KEY = '任意一串字母数字，比如 abc123xyz'
```
随便写一串字母数字就行，记住这串内容。

4. 第二行 `IDATA_COOKIE` 先不用动（保持原样），这一步可能需要联系发给你这个软件的人，问他要他的 iData Cookie。

5. 保存文件，关闭。

---

## 七、启动软件

在终端里输入下面这行，按回车：

```
python3 app.py
```

（Windows 如果提示 python3 不认识，就输入 `python app.py`）

你会看到终端显示类似这样的内容：
```
* Running on http://127.0.0.1:5000/
* Running on http://localhost:5000/
```

**看到这个就说明软件启动成功了！** 不要关闭这个终端窗口。

## 九、常见问题

### Q：终端显示「No module named 'flask'」

说明上一步安装依赖没成功。重新在终端里运行：
```
pip3 install -r requirements.txt
```
确保看到 `Successfully installed`。

### Q：浏览器打不开 http://localhost:5000

确认终端里还在运行 `python3 app.py`，那个窗口不能关。

### Q：提示密码错误

默认账号是 `admin` / `admin123`，注意都是小写。

### Q：登录进去后看不到数据/拉取数据报错

这是因为 `config.py` 里的 `IDATA_COOKIE` 还没改成你自己的。需要联系软件负责人（发给你的人），让他告诉你他的 iData Cookie，你替换进去。

### Q：Windows 上 pip 命令找不到

重新安装 Python，安装时**一定要勾选**「Add Python to PATH」。

## 十、以后每次怎么启动

以后想用这个软件，步骤是：

1. 打开终端，进入项目目录（第四步）
2. 输入 `python3 app.py`（第七步）
3. 浏览器打开 http://localhost:5000（第八步）
4. 用完关闭终端窗口即可
