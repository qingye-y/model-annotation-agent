## 一、审核提示词体系总览

### 1.1 审核分类结构

```
商品名称
├── 无关词语违规
├── 品类词堆砌
├── 标题与图片不一致
└── 标题与销售属性不符

商品主图
├── 主图非主体展示（第1张）
├── 多主体/主体不明
├── 违规引流信息
├── 水印信息
├── 品牌旗舰店
└── 商品清单

商品详情图
├── 多主体/主体不明
├── 违规引流信息
├── 水印信息
├── 品牌旗舰店
└── 商品清单

商品SKU图
├── SKU图与商品信息不一致
└── SKU图含商品清单

类目
├── 党徽/党旗/国徽/国旗禁售
├── 当前网超限制售卖
└── 类目与实际不符
```

### 1.2 核心审核规则速查

| 规则类型 | 违规触发条件 | 核心关键词 |
|---------|------------|-----------|
| 标题无关词 | 标题含无意义字符串/繁体/生僻字/emoji | 意义、字符、字符串、生僻字、繁体字 |
| 品类词堆砌 | 标题含多个与主体无关的品类词 | 堆砌、混放 |
| 标题-图片不一致 | 标题与主图品类/品牌/型号不符 | 一致、不符、实际、标题、主体 |
| 标题-属性不符 | 销售属性（颜色/数量/型号）与标题不符 | 属性、销售、数量、颜色 |
| 主图非主体 | 主图第1张未以售卖主体为主 | 未以、为主、主体 |
| 多主体违规 | 主图/详情图含多品类商品 | 多个、主体、不明 |
| 违规引流 | 图片含京东/淘宝/拼多多等平台引流 | 引流、京东、淘宝、天猫、商城、旗舰店 |
| 水印违规 | 图片含禁用水印文字 | 水印、商贸、商城、贸易、科技 |
| 品牌旗舰店 | 图片含"旗舰店"/"专卖店"等 | 旗舰店、专卖店、专营店 |
| 商品清单 | 图片含Excel格式商品清单 | 清单、表格 |
| SKU不符 | SKU图品牌/型号/颜色与主图不一致 | 张图、sku |
| 党徽国旗 | 商品含党徽/党旗/国徽/国旗且无资质 | 国旗、党徽、国徽、党旗、资质、批准、文件 |
| 限制售卖 | 含黑名单品类关键词 | 限制、当前、网超、禁售、需提供、相关 |
| 类目不符 | 商品与末级类目不匹配 | 类目、末级、匹配、相关 |

## 四、关键词提取方法与泛化词列表

### 4.1 关键词提取流程

```
原始 reject_reason 文本
    ↓
fix_encoding() — latin1 → UTF-8 解码（解决iData编码问题）
    ↓
jieba 分词
    ↓
正则过滤规则：
    ├─ 过滤纯ASCII数字/字母/符号（保留含汉字的词）
    ├─ 过滤含泛化停用词的 3-4 字词（避免误杀2字业务词）
    └─ 保留含汉字的业务关键词
    ↓
Counter 统计频率
```

### 4.2 泛化停用词（关键词提取时已过滤）

| 类别 | 停用词列表 |
|------|----------|
| 审核动作 | 发布、图文、详情、售卖、发布、出现、补充 |
| 信息状态 | 一致、不符、相关、匹配、实际、不明 |
| 图片描述 | 主体、为主、未以、多个、张图、sku |
| 通用词 | 意义、数量、需要、没有 |

### 4.3 保留的有效关键词特征

- 含汉字（防止纯ASCII数字/字母被误判为业务词）
- 长度 ≥ 5 字（短词已通过泛化词过滤）
- 非纯品牌词（非2字常见品牌）
- 非度量单位词

## 七、使用说明

### 7.1 关键词 → 提示词溯源

当发现某个关键词高频出现时，按以下路径溯源到具体提示词规则：

```
关键词
    ↓
判断违规类型（参照 第二章 映射表）
    ↓
定位到 提示词_V24 中对应的审核分类和审核项
    ↓
查看该审核项下的详细规则、示例和豁免条件
```

### 7.2 提示词 → 关键词验证

当审核规则调整后，可通过以下方式验证效果：

```
调整提示词规则
    ↓
重新在5个实例上运行探查脚本
    ↓
对比关键词频次变化
    ↓
判断规则调整是否生效
```
